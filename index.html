GLOBAL NEWS SOURCE

ABOUT GLOBAL NEWS SOURCE

Global News Source is a news filtering machine designed to source relevant news articles in an easy and functional way. Pick your country and category of choice and add a keyword to further filter your search. Global News Source uses NewsAPI as a database, News API is a simple HTTP REST API for searching and retrieving live articles from all over the web. This allows the search outcomes to be filtered form every single article posted online. This web applictaion was made as a project for learning purposes.
